# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project you agree to abide by its terms.

## Reporting

To report a concern, please contact the maintainers privately through GitHub
(Security → Report a vulnerability on `imanect-labs/agent-start`, or by
direct message to a maintainer listed on the organization page). All reports
will be reviewed and investigated promptly and fairly.

The full text of the Contributor Covenant is available at the link above and
is incorporated here by reference.
